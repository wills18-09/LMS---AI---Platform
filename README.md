# LMS AI Platform

An AI-powered Learning Management System (LMS) built using Node.js, Express.js, TypeScript, PostgreSQL, Redis, and Docker.

## Features Completed

- User Registration & Login
- JWT Authentication
- Role-Based Access Control (Student, Instructor, Admin)
- Course CRUD APIs
- Course Enrollment
- PostgreSQL Database Schema
- Docker Setup

## Tech Stack

- Node.js
- Express.js
- TypeScript
- PostgreSQL
- Redis
- Docker
